# 🚀 Guia de Início Rápido

Comece a usar o Finance Manager Pro em menos de 5 minutos!

## ⚡ Quick Start

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/finance-manager-pro.git
cd finance-manager-pro

# 2. Instale as dependências
npm install

# 3. Inicie o projeto
npx expo start

# 4. Escaneie o QR Code com Expo Go
```

## 📱 Primeiros Passos

### 1. Escolha o Tipo de Conta

Na primeira vez que abrir o app:

- **Pessoa Física (PF)**: Para uso pessoal
- **Pessoa Jurídica (PJ)**: Para empresas

💡 Você pode mudar isso depois nas configurações

### 2. Adicione sua Primeira Transação

1. Toque no botão **+** (canto inferior direito)
2. Escolha o tipo: **Receita** ou **Despesa**
3. Preencha:
   - Descrição (ex: "Salário", "Supermercado")
   - Valor (ex: R$ 5.000,00)
   - Categoria (ex: "Salário", "Alimentação")
   - Data
4. Toque em **Salvar**

### 3. Visualize seu Dashboard

O dashboard mostra:
- 💰 **Saldo total**
- 📈 **Receitas do mês**
- 📉 **Despesas do mês**
- 📊 **Gráficos e análises**

### 4. Filtre e Busque Transações

- Use a barra de busca para encontrar transações específicas
- Filtre por tipo (receita/despesa)
- Filtre por categoria
- Filtre por período

### Criar Categoria Personalizada

1. Vá em **Configurações** > **Categorias**
2. Toque em **Nova Categoria**
3. Escolha:
   - Nome
   - Ícone
   - Cor
   - Tipo (receita/despesa)
4. Salvar

## 🏦 Conectar Banco (Open Finance)

> ⚠️ Requer configuração adicional - veja [OPEN_FINANCE_GUIDE.md](./docs/OPEN_FINANCE_GUIDE.md)

1. Vá em **Configurações** > **Contas Bancárias**
2. Toque em **Conectar Banco**
3. Escolha sua instituição financeira
4. Autorize o acesso
5. Aguarde sincronização

### Benefícios
✅ Transações importadas automaticamente  
✅ Saldo atualizado em tempo real  
✅ Categorização inteligente  
✅ Múltiplas contas em um lugar

## 💡 Dicas e Truques

### 1. Use Descrições Claras
❌ "Compra"  
✅ "Supermercado Extra - Compra Mensal"

### 2. Categorize Consistentemente
- Sempre use a mesma categoria para o mesmo tipo de gasto
- Exemplo: Sempre use "Alimentação" para supermercado

### 3. Adicione Notas
- Use o campo de notas para detalhes importantes
- Exemplo: "Parcelado em 3x" ou "Incluiu frete R$ 15"

### 4. Marque como Não Pago
- Para transações futuras ou parceladas
- Ajuda a saber o que ainda precisa pagar

### 5. Revise Semanalmente
- Dedique 10 minutos por semana
- Confira se todas as transações foram registradas
- Analise seus gastos

## 🔧 Configurações Recomendadas

### Para Uso Pessoal (PF)
```
Tipo de Conta: Pessoa Física
Moeda: BRL (R$)
Notificações: Ativadas
Sincronização: Diária
```

### Para Uso Empresarial (PJ)
```
Tipo de Conta: Pessoa Jurídica
Moeda: BRL (R$)
Categorias: Ativar categorias PJ
Backup: Semanal
```

## ❓ Perguntas Frequentes

### Como editar uma transação?
Toque na transação na lista → Faça as alterações → Salvar

### Como deletar uma transação?
Toque e segure na transação → Confirme exclusão

### Posso usar em múltiplos dispositivos?
Atualmente os dados ficam locais. Sincronização em nuvem está planejada para v2.1

### É gratuito?
Sim! Totalmente gratuito e open source.

### Meus dados são seguros?
Sim. Tudo fica armazenado localmente no seu dispositivo. Sem servidores externos.

### Funciona offline?
Sim! O app funciona 100% offline. Open Finance requer internet.
